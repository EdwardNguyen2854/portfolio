## Overview
This portfolio was built to demonstrate clean code and minimalist design principles. It uses no external frameworks, relying purely on semantic HTML, CSS variables for theming, and simple JavaScript for interactivity.

## Key Features
- **Design Tokens**: Centralized color and spacing management.
- **Client-side Search**: Filter projects and wiki entries instantly.
- **Responsive**: Adapts perfectly to mobile and desktop screens.
